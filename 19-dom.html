//difference between textContent and innerText and innerHTML if we use textContent, it will return all the text content of the element, including hidden elements and script/style tags. It does not consider the CSS styling and will return the text as it is in the DOM. if we use innerText, it will return only the visible text content of the element, excluding hidden elements and script/style tags. It considers the CSS styling and will return the text as it is displayed on the page. if we use innerHTML, it will return the HTML content of the element, including any HTML tags and content. /* title.textContent 'DOM learning on Chai aur code test text' title.innerText 'DOM learning on Chai aur code' title.innerHTML 'DOM learning on Chai aur code test text' */ //QuerySelector and QuerySelectorAll //querySelector will return the first element that matches the specified CSS selector(s) in the document //querySelectorAll will return a static NodeList of all elements that match the specified CSS selector(s) in the document //example: /* const firstHeading = document.querySelector('h1'); const allHeadings = document.querySelectorAll('h2'); console.log(firstHeading); // returns the first h1 element console.log(allHeadings); // returns a NodeList of all h2 elements */ //node list is a collection of nodes, which can be elements, text nodes, or comments.
